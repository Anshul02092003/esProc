dyna_query()

Description:

Query data stored in DynamoDB.

Syntax:

dyna_query(dyna_open, sql)

Note:

The external library function queries data in DynamoDB using a SQL statement and return result as a table sequence.

Parameters:

dyna_open

The DynamoDB connection object

sql

A SQL statement

Return value:

A table sequence

Example:

 

A

 

1

=dyna_open("http://localhost:8000", "us-east-2")

 

2

=dyna_query(A1, "select * from Music")

 

3

=dyna_query(A1, "select * from Music where Artist between ? and ?", "Hello", "No One You Know" )

 

4

=dyna_query(A1, "SELECT * FROM Music where Artist=? and SongTitle=?", "Acme Band", "Happy Day")

 

 

5

>dyna_close(A1)
