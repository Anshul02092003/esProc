k.len()

Description:

Get the length of a serial byte.

Syntax:

k.len()

Note:

The function returns the length of serial byte k, which is the number of bytes in k.

Parameters:

k

A serial byte

Return value:

An integer

Example:

k(2).len()

1

k(256:2).len()

2

k(1,2,3).len()

3
