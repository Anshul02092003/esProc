redis_get()

Description: 

Get the values of all the given keys.

Syntax:

redis_get(con,key[,key1,key2,…])

Note:

This external library function supports getting the key value or values of multiple keys.

Parameters:

con

Database connection string

key

Key name

Return value:

  String

Example:

 

A

 

1

=redis_open()

 

2

=redis_get(A1,"name")

3

=redis_get(A1,"id","name","age")
