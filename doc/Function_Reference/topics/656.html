redis_ltrim()

Description:

Trim a list table to the specific size.

Syntax:

redis_ltrim(con,key,start,end)

Note:

The external library function trims a list to retain elements in a specified interval while deleting the other elements not included in the interval.

The interval is defined by two offsets, start and end (both inclusive). 0L represents the first element. A negative offset means counting from backwards, such as -1L, which is the last element, and -2L, which is the second-to-last.

Parameters:

con

Database connection string

key

List table name

start

The starting position, which is the long type

end

The ending position, which is the long type

Return value:

Boolean

Example:

 

A

 

1

=redis_open()

 

2

=redis_lrange(A1,"mylist",0,10)

3

=redis_ltrim(A1, "mylist",0L,2L)

 

4

=redis_lrange(A1,"mylist",0,10)
