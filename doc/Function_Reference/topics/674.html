redis_zscan()

Description:

Scan members in in ordered set iteratively.

Syntax:

redis_zkeys(con,key,regex,[count])

Note:

The external library function scans members (including element and values under each element) in an ordered set iteratively and stores the data in a corresponding already defined data structure. Error will be reported if the data strucrue does not exist.

Parameters:

con

Database connection string

key

Name of the ordered set

regex

A regular expression

count

The number of key values scanned at a time

Return value:

A table sequence

Example:

 

A

 

1

=redis_open()

 

2

=redis_zscan(A1, "runoobkey", " *", 3)
