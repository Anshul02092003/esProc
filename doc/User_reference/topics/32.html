Auto Backup (.BAK)

Whether or not to automatically save the original file when you save a modified esProc file (.dfx/.splx).

A check of the option will enable the system to automatically save a backup copy of the unmodified file when you save the modified file. This lets a user to recover the file using the backup copy whenever there is something wrong with the modification. The system won¡¯t automatically save a backup copy of the original file when a modified copy is saved if the option is unchecked.
