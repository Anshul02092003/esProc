Log level

It specifies the logging level of Backstage Print.

The logging levels include OFF, SEVERE, WARNING, INFO, DEBUG and ALL, whose priorities descend in the listed order. With the OFF level, no log information will be exported; with INFO level, only information at the levels of ERROR, WARNING, INFO will exported. The rest levels can be inferred.
