12.6      The Line Element

The Dot Element discussed how to position a chart element using logical coordinates. Here let’s look at the line element. The line element is another one of the most basic chart elements, which is positioned by at least two pairs of logical coordinates. If there are more logical coordinates, multiple lines can be plotted continuously according to them.
