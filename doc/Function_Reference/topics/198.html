env()

Description:

Assign value to the global variable.

Syntax:

env(v,x)

Note:

The function assigns value x to the global variable v. It will lock v when calculating x to make sure the global variable won’t be changed dureing the process. With x omitted, the global variable will be deleted.

Parameters:

v

The global variable

x

Variable value

Options:

@j

Set the global variable v as a task variable

Return value:

The variable value or null

Example:

 

A

 

1

=env(arg1,12)

12. Assign 12 to the global variable arg1

2

=env(arg1)

Delete the global variable arg1 as the variable value is omitted

3

=env@j(arg2,3)

Set the global variable arg2 as a task variable
