k1+k2

Description:

Concatenate serial bytes.

Syntax:

k1+k2

Note:

Concatenate serial byte k1 and serial byte k2; both k1 and k2 must be serial bytes.

Parameters:

k1

A serial byte

k2

A serial byte

Return value:

A serial byte

Example:

k(1)+k(3:2)

100030000000000

k(1:2)+k(1,3,4)

1010304000000
