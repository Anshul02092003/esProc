2.21       Webservice

1. The functioning of WebserviceCli external library requires the embedded third-party jar: jdom-1.0.0.jar. The jar is located in esProc class path: installation directory\esProc\lib.

 

2. The Raqsoft core jar for this external library is scu-webservice-cli-2.10.jar. The directory containing files of this external library is: installation directory\esProc\extlib\WebserviceCli.

 

3. esProc provides two external library functions - ws_client() and ws_call() - to access multidimensional databases. Look them up in【Help】-【Function reference】to find their uses.
