lower()

Description:

Convert all characters of a string to lower case.

Syntax:

lower(s)

Note:

The function converts all characters of a string to lower case.

Parameters:

s

The string you want to convert to the lower case

Options:

@q

A quoted string won’t be converted

Return value:

A string

Example:

lower("ABCdef")

abcdef

lower("defABC")

defabc

lower@q("\'AD\'")

lower("\'AD\'")

'AD'

'ad'
