Description:

Syntax:

Note:

The function associates cursor cs and cursor csi/record sequence Ai through the join key and forms a sequence consisting of x:F,¡­ and xi:Fi,¡­ to return as a cursor; by default, x:F,¡­ involves all fields of cs. Both cs and csi/Ai should be ordered by the join key.

Parameter z specifies the join type. It can be absent or null. Perform an inner join when z is absent, and a left join when z is null; and only retain records of cs that cannot find matches when z is null and both parameters xi:Fi are absent.
