A.ifn()

Description:

Get the first non-null member of a sequence.

Syntax:

A.ifn()

Equivalent to ifn(x1,бн,xn)

Note:

The function gets the first non-null member of sequence A.

Parameter:

A

A sequence with the length of n

Return value:

The first non-null member of the sequence.

Example:

 

A

 

1

=[1,2,4].ifn()

1

2

=[null,2,3,4].ifn()

2

3

=[null,2,null,4].ifn()

2

4

=ifn(null,1,2,4)

1
