datetime(datetimeExp)

Description:

Adjust the precision of the datetime expression and return it.

Syntax:  

datetime(datetimeExp)

Note:  

The function adjusts the precision of datetimeExp and returns it. By default, the precision is the day.

Parameter:  

datetimeExp

Datetime value

Options:

@m

Measure to minute

@s

Measure to second

Return value:  

Datetime value

Example:

datetime(now())

2013-12-09 00:00:00

datetime@m(now())

2013-12-09 16:56:00

datetime@s(now())

2013-12-09 16:56:45

Related functions:

date()
