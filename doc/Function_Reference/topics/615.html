redis_expire()

Description:

Set expire time for a specified key.

Syntax:

redis_expire(con,key,timeout,[timeunit])

Note:

The external library function sets the key¡¯s expire time. The key is invalid after it is expired.

Parameters:

con

Database connection string

key

Key name

timeout

Expire time, whose default unit is second when the field is long type; parameter timeunit is not needed when the field is data type or when values of the field are data strings of yyyy-MM-dd format

timeunit

Time unit specified by values corresponding to different unit:

0: nanosecond; 1: microsecond; 2: milisecond; 3: second; 4: minute; 5: hour; 6: day

Return value:

Boolean

Example:

 

A

1

=redis_open()

2

=redis_expire(A1,"name",20)

3

=redis_expire(A1,"name",10000,2)

4

=redis_expire(A1,"name","2000-01-01")

5

=redis_expire(A1,"name",data(2000-01-01))
