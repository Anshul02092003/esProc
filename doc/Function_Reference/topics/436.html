ifnumber()

Description:

Find whether the parameter is of numeric data type.

Syntax:

ifnumber(Exp)

Note:

The function finds whether the parameter Exp is of numeric data type.

Parameter:

Exp

Expression of any data type

Return value:

Boolean value

Example:

ifnumber("abc")

false

ifnumber("1234")

false

ifnumber(1234)

true

ifnumber("1234sss")

false
