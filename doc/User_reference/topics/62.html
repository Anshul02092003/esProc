Search path

It specifies the relative paths for referencing some files by the cellset program, for example, the path used in the file function. Users need to look up the relative path list to locate files in some cases. There can be multiple Search Paths, separated by semicolons.

For a relative path, the directory to which it is linking side is [installation directory]/esProc on IDE, and the application root path on web side.

Search files in non-absolute paths for a desired file according to the specified order: class path-search path-main directory.
