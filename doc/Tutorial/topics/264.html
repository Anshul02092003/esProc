11.8      A sketch of DQL

In order to make queries on pseudo tables, paticularly those involving multi-table joins, more conveniently, esProc designs DQL tool to perform queries using Dimensional Query Language (DQL). Here we just give a priliminary introduction to the DQL tool. You can find detailed explanations in esProc DQL Tutorial.
